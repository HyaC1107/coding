## 1. 공통

인증 및 전 회원 타입이 공유하는 기능입니다.

### 1-1. 인증

| 기능 | 경로 | 상태 | 비고 |
|------|------|:----:|------|
| 로그인 | `/` | ✅ | 고객 / 파트너 탭 전환, POST `/auth/login` |
| 고객 회원가입 | `/signup` | ✅ | 아이디 중복확인, 비밀번호 확인, POST `/auth/register` |
| 파트너 회원가입 | `/signup/partner` | ✅ | 시공업체 / 중장비업체 선택, 서류 파일 업로드, POST `/auth/register/partner` |
| 아이디 / 비밀번호 찾기 | `/find-account` | ✅ | 이름+전화+이메일 조회, 결과 화면 표시, POST `/auth/findId` · `/auth/findPw` |

### 1-2. 레이아웃 (TopNavLayout)

| 기능 | 상태 | 비고 |
|------|:----:|------|
| 상단 유틸리티 바 (알림 / 로그인 / 로그아웃 / 앱다운로드) | ✅ | |
| 알림 벨 클릭 → 알림 드롭다운 모달 | ✅ | |
| 회원 타입별 내비게이션 메뉴 | ✅ | 활성 탭 하이라이트 |
| 주소 선택바 | ✅ | 업체찾기(`/customer/main`) · 시공PICK(`/customer/pick`) 페이지에서만 표시, 다음 주소 API |

### 1-3. 공통 페이지

| 기능 | 경로 | 상태 | 비고 |
|------|------|:----:|------|
| 알림 목록 | 모달 내부 | 🟡 | 날짜별 그룹, 리뷰 알림 시 리뷰쓰기 버튼, GET `/notifications` · PATCH `/notifications/:id/read` (더미 폴백) |
| 공지사항 | `/{type}/notice` | 🟡 | 공지 / 이벤트 탭, 아코디언 펼침, GET `/notices` (더미 폴백) |
| 1:1 문의하기 | `/{type}/qna` | ✅ | 문의하기 탭(카테고리·제목·내용), 문의내역 탭(아코디언·상태 배지), POST `/qna` · GET `/qna` |
| 채팅 목록 | `/{type}/chat` | 🟡 | 카테고리 탭 24개, 읽지 않음 뱃지, GET `/chat/rooms` (더미 폴백) |
| 채팅방 | `/{type}/chat/:roomId` | ✅ | Socket.io (joinRoom · sendMessage · receiveMessage · leaveRoom), 메시지 초기 로드 GET `/chat/:roomId/messages`, + 메뉴 (사진첨부 · 견적서 · 대화방나가기) |
| 리뷰관리 | `/{type}/reviews` | 🟡 | 작성한 리뷰 / 미작성 리뷰 탭, 별점 표시, GET `/reviews/user/:userId` (더미 폴백) |
| 회원정보 수정 | `/{type}/profile/edit` | 🟡 | 비밀번호 인증 → 수정 폼(아이디 읽기전용 · 비밀번호 · 이메일 · 닉네임 중복확인), PATCH `/users/:userId` (비밀번호 인증 엔드포인트 없음) |

## 2. 고객 (Customer)

### 2-1. 업체찾기 `/customer/main`

| 기능 | 상태 | 비고 |
|------|:----:|------|
| 배너 캐러셀 | 🟡 | 더미 이미지 |
| 주거 / 상가 토글 | 🟡 | UI만 존재, API 필터 미연동 |
| 공종 카테고리 그리드 32개 (펼쳐보기 토글) | ✅ | 클릭 시 시공PICK으로 이동 |

|------|:----:|------|

| Step 2 — 업체 선택 최대 3개 | 🟡 | GET `/companies`, 선택 뱃지, 더미 폴백 |

| 주소 선택바 | ✅ | |

### 2-3. 스케줄 `/customer/schedule`

| 기능 | 상태 | 비고 |

| 희망방문요청 탭 | 🟡 | GET `/requests`, 더미 폴백 |

### 2-4. 마이페이지 `/customer/mypage`

| 기능 | 상태 | 비고 |

## 5. 미구현 항목 요약

| 항목 | 해당 타입 | 비고 |
|------|-----------|------|
| 스케줄 안내문 / 스케줄 관리 버튼 | 고객 | 기능 정의 필요 |
| 주거/상가 토글 API 필터 연동 | 고객 | 시공PICK · 업체찾기 |
| 우리가게 꾸미기 (업체 소개 편집) | 시공업체 | 백엔드 미구현 |

| 회원정보 수정 — 비밀번호 인증 | 전체 | POST `/auth/verify-password` 엔드포인트 없음 |
